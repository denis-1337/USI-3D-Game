# Original USI - Star Fleet DOS Source Code

This directory contains the original source code and assets from the **USI - Star Fleet** DOS game by **Ben Becker**.

## Files

| File | Description |
|---|---|
| `USI.PAS` | Main game program (Turbo Pascal) |
| `PLANETEN.PAS` | Planet logic and combat |
| `AS.PAS` | Artificial intelligence |
| `BILD.PAS` | Graphics and rendering |
| `CONSTANT.PAS` | Game constants |
| `MOUSE.PAS` | Mouse input handling |
| `VEC1.PAS` | Vector math utilities |
| `EGAVGA.BGI` | VGA graphics driver |
| `usi.txt` | Original game documentation |
| `VEC1.o` / `VEC1.ppu` | Compiled Pascal units |

## Compiler

The original source was written for **Turbo Pascal** and targets the **EGA/VGA** graphics mode on DOS.
